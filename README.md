# Grid Card

> 一个围绕数据驱动、确定性随机与事务化触发编排构建的 Unity 卡牌游戏。

**Grid Card** 的工程核心是一套可配置、可重放、可回滚的九宫格触发系统。项目重点处理复杂效果的执行顺序、异步生命周期、随机复现和运行时状态一致性。

`Unity 2022.3.60f1c1` · `C#` · `URP 14` · `VContainer 1.17.0` · `Addressables 1.22.3` · `UniTask 2.5.10`

## 技术核心

### 触发编排

`TriggerManager.StartTrigger()` 是唯一编排入口。每轮按九宫格顺序推进，并在同一条流程中协调卡牌、修饰词条、递归触发、标签、牌型、Boss 规则和关卡结果。

```text
遍历格子
  → 前置词条
  → 卡牌内容 / 递归触发
  → 后置词条
  → 卡牌与格子标签
  → 牌型倍率与窗口分结算
  → 关卡结果
  → 提交或整体回滚
```

- 额外触发会完整等待目标格返回，并通过递归链检测防止循环。
- 词条具有前置/后置时机和局部/全局生命周期，队列顺序就是实际执行顺序。
- 规则状态、关卡会话和 UI 投影分离，界面不持有玩法权威。
- 触发前同时快照存档、关卡会话、棋盘、牌堆和随机进度；流程异常或取消时整体恢复，完整成功后才提交。

### 配置链路

```text
XLSX → CSV → DefinitionSO / LibrarySO → Manager / Runtime
```

- `GridCard配置.xlsx` 是人工编辑权威，CSV 是导入快照，ScriptableObject 是运行时投影。
- 数值、权重、文案和组合由配置维护；执行语义由代码中的强类型处理函数负责。
- 配置通过最终定义 ID 进入 Registry，同一份 metadata 同时服务于参数校验、资源校验和运行时执行。
- Addressables 使用资源键加载配置、Prefab 与面板；Player Build 会自动重建 Addressables Content。

### 存档与确定性随机

- **存档聚合根**：`ArchiveState` 负责 JSON 与快照结构，`ArchiveRuntimeManager` 负责加载、规范化和明确写入口，领域系统只通过窄 Store 读写自己的状态。
- **稳定保存边界**：进关、离店、关卡结算等关键阶段统一保存；`LoadRevision` 与 operation owner 阻止旧异步流程覆盖新存档。
- **独立随机流**：由 `MasterSeed + 稳定流 Key` 派生牌库、商店、Boss、奖励和效果等随机流，互不干扰。
- **可重放进度**：存档只记录各随机流的 `DrawCount`；读档后从同一边界恢复，可重现洗牌、候选与效果结果。

### 工程架构

| 方向 | 实现 |
| --- | --- |
| 程序集分层 | `Game.Core`、`Game.ScriptableObject`、`Game.Component`、`Game.Manager`、`Game.App` |
| 依赖注入 | VContainer + Stratum，通过 `GameLifetimeScope` 与 `ManagerOrder` 统一注册运行时服务 |
| 异步流程 | UniTask 串联启动、资源、面板、牌堆和触发流程，并结合取消与版本校验管理生命周期 |
| 资源管理 | Addressables 统一资源定位、异步加载和句柄缓存，业务代码不依赖具体资源路径 |

调用方按职责依赖窄接口；静态配置、运行时状态、磁盘存档与 UI 展示分别维护，避免同一状态出现多个权威来源。

## 游戏玩法

Grid Card 是一款在九宫格中编排卡牌、按环形顺序触发效果，并用扑克牌型放大得分的单机牌组构筑游戏。

棋盘编号与默认结算顺序：

```text
1   2   3
8   9   4
7   6   5

1 → 2 → 3 → 4 → 5 → 6 → 7 → 8 → 9
```

1. 拖拽交换九宫格中的卡牌，规划触发顺序。
2. 触发牌提供基础分并进入牌型队列；对子、顺子、同花等牌型会放大当前窗口分。
3. 修饰牌添加词条或递归触发其他格子，功能牌用于换位、改牌、控制牌型和操作商店。
4. 每关初始拥有 3 次触发与 3 次弃牌；在触发耗尽前达到目标分即可通关。

普通关通关后进入商店，每 3 关出现一次 Boss。击败 Boss 后可在长期事件与固定金币之间选择奖励，完整流程共 24 关。

当前内容包含 **52 张基础手牌、50 张触发牌、50 张修饰牌、50 张功能牌与 15 种牌型**。

下载仓库中的 Windows 构建版本，解压后运行 `Grid Card.exe`。试玩不需要安装 Unity。
